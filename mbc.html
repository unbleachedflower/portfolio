My Building Code
Desktop, Tablet, and Mobile

Overview

Ontario Building Code is the legislative framework governing the construction, renovation and change-of-use of a building in the Province of Ontario. The existing building code is only available in a paper format. It is presented as a two-volume set of 7-ring binders.

01  Problem

The existing building code is only available in a paper format. It is presented as a two-volume set of 7-ring binders. Transporting these binders from office to construction fields is cumbersome as they weigh close to 40 lbs altogether.

To keep up with the ever-changing building code and make sure that they are referencing the documents properly, professionals need to attend special sessions to learn how to navigate the pages.

02  Competitive Analysis

Most of the competitors' sites we encountered have three significant problems:

1 | Still presenting old versions of the code, information is not up to date.
2 | Old school interface designer that is very difficult for users to navigate around the site and find their desired informatio.
3 | Design is not responsive and cannot be access easily.

06  Wireframes

Three different set of wireframes were designed simultaneously for desktop, tablet, and mobile end users. The wireframes define key functionality such as bookmark, new tabs, share, and comment. We also incorporated an occupant loader calculator to help users calculate their construction loading capacity.
